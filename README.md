## power_bi_analyst: Relatório de Vendas Elegante com Power BI

### Sobre o projeto
Este projeto é parte do bootcamp da Dio, que abrangeu **Python avançado** e **Power BI**. O objetivo é criar um relatório refinado com dados financeiros no Power BI, usando arquivos disponíveis [aqui](https://github.com/julianazanelatto/power_bi_analyst). O projeto inclui botões de navegação, segmentadores, indicadores e uma segunda página para uma experiência completa e interativa, proporcionando uma ferramenta robusta para análise de dados financeiros.

O objetivo deste bootcamp é criar um relatório claro e organizado, implementando botões de navegação, segmentadores e indicadores. Também será adicionada uma segunda página para proporcionar uma experiência de usuário completa.
